# BesovSpacesGoodGrid

This Lean 4 project is in its initial stages. We aim to formalize the main result on Besov spaces for
measure spaces with good grids, as defined in

Smania D.
Besov-ish spaces through atomic decomposition. Analysis & PDE Vol. 15 (2022), No. 1, 123-174
DOI: https://doi.org/10.2140/apde.2022.15.123


The definition of Unbalanced Haar Wavelets and the main results about them (https://github.com/SmaniaD/BesovSpacesGoodGrid), already formalized in another project , will play an important role here.
