# Preprint Source

This folder contains the LaTeX source file for the preprint

Daniel Smania, *Besov-ish spaces through atomic decomposition*,
available at <https://arxiv.org/abs/1903.06937>.

This Lean repository is based on that preprint and formalizes selected parts
of its theory of Besov-ish spaces, atomic decompositions, weak grids, good
grids, Souza atoms, and related norm comparisons.
