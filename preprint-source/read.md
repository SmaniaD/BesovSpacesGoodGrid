# Preprint Source

This folder contains the LaTeX source file for  

Daniel Smania, *Besov-ish spaces through atomic decomposition*,
available at <https://arxiv.org/abs/1903.06937>.

The printed version appeared as:

Daniel Smania, *Besov-ish spaces through atomic decomposition*,
*Analysis & PDE* 15 (2022), no. 1, 123-174.
DOI: <https://doi.org/10.2140/apde.2022.15.123>.

This Lean repository is based on that source and formalizes significant  parts of
its theory of Besov-ish spaces, atomic decompositions, weak grids, good grids,
Souza atoms, and related norm comparisons.
